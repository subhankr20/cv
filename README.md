# PATRA: THE PITCH

A 2.5D side-scrolling interactive portfolio.

## Stack
- React 18 + Vite
- Tailwind CSS
- GSAP + ScrollTrigger
- @pixi/react
- Lenis (smooth scroll)
- Howler.js

## Run locally
1. `npm install`
2. `npm run dev`

## Deploy
Run `npm run deploy` to push the build to the `gh-pages` branch.
